EVENTS

BACKGROUND

Cineplex Events (formerly Front Row Centre Events) bring world class entertainment to local theatres. Presented in HD with digital surround sound, guests can experience the best in original one-night only and series-based programming (Met Opera Live in HD, Broadway productions, London’s National Theatre broadcast live direct from the stage, dance performances, classic films, concerts, eSports, live sporting events and documentaries).

URLs

The Events URL should be used when an ad combines various programs into one piece. Otherwise, one of the following URLs should be used where appropriate for each ad:

Programs: National Theatre Live, Branagh Theatre Live, Stratford Festival HD
URL: Cineplex.com/Stage

Program: The Metropolitan Opera HD Live
URL: Cineplex.com/MetOpera

Programs: In The Gallery
URL: Cineplex.com/InTheGallery
